This position will be part of Software and Advanced Technology ( S A T) Security Software ( S W) organization. This group is responsible for developing scalable security software products that creates value and simplifies integration of Intel Architecture ( I A) security technologies in the cloud as this is where security platform features are manifested. This specific position will be to design, develop and test additional feature sets for the S W product to secure key cloud infrastructure data.
